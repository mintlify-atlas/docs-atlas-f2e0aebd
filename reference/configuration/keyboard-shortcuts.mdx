---
title: Keyboard shortcuts
description: Keyboard shortcuts for common actions in ApiArk.
---

<Note>
  Keyboard shortcuts can be customized in **Settings → Keyboard Shortcuts**. The shortcuts listed here are the defaults.
</Note>

On macOS, `Ctrl` is replaced by `Cmd` for most shortcuts.

## General

| Shortcut | Action |
|---|---|
| `Ctrl+K` / `Cmd+K` | Open command palette |
| `Ctrl+,` | Open settings |
| `Ctrl+Shift+P` | Toggle sidebar |

## Tabs

| Shortcut | Action |
|---|---|
| `Ctrl+T` | New request tab |
| `Ctrl+W` | Close current tab |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Ctrl+1` through `Ctrl+9` | Switch to tab by number |

## Requests

| Shortcut | Action |
|---|---|
| `Ctrl+Enter` | Send request |
| `Ctrl+S` | Save request |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+D` | Duplicate request |

## Collections

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+N` | New collection |
| `Ctrl+O` | Open collection folder |
| `Ctrl+Shift+F` | Search in collection |

## Panels

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+C` | Open console |
| `Ctrl+Shift+H` | Open history |
| `Ctrl+Shift+E` | Open environment editor |
